# Report

## Task 2

### Changes to Task 1

-   Added to new macros to replace the `PRODUCT_SIZE` macro:
    -   `PRODUCT_MIN_SIZE` - The minimum size of a product
    -   `PRODUCT_MAX_SIZE` - The maximum size of a product
-   When creating a product, the size is now randomly generated between `PRODUCT_MIN_SIZE` and `PRODUCT_MAX_SIZE` instead of being a predefined constant.
